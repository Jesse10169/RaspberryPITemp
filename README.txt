- About 
    - These are three phyton scripts that work independently from each other but they all have an essential function.
    The program Temp_ program.py is in charge of gathering information from the DHT11 sensor using the rasoberry pi.
    The program Program_Graph.py takes the information from Stored_Data.csv stored by Temp_ program.py and plots and creates
    a graph. Lastly the program Stored_SQLite.py takes the data from Stored_Data.csv and stores it to Cloud_Data.db which
    is manually uploaded to a free cloud base service.
